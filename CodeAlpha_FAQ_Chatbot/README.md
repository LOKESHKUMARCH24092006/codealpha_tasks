# 💬 FAQ Chatbot

## 📌 Overview
This project is an AI-powered FAQ Chatbot that answers user questions by matching them with the most relevant FAQ from a predefined dataset. It uses Natural Language Processing (NLP) techniques and cosine similarity to provide accurate responses.

## 🚀 Features
- Answer frequently asked questions
- NLP-based text preprocessing
- Question similarity matching
- Fast and accurate responses
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python
- NLTK / SpaCy
- Scikit-learn
- Pandas
- Streamlit

## ⚙️ How It Works
1. Collect FAQs and answers.
2. Preprocess the text using NLP techniques.
3. Convert text into numerical vectors.
4. Compare user queries with FAQs using cosine similarity.
5. Return the best matching answer.
